README

To run the HEV P2 model, please do the following:

2.      Extract HEVP2_R2018a.zip file
3.      Set your MATLAB Current Folder to:  HEVP2_R2018a / main
4.      Double click on HEVP2.prj.  This Simulink project file will launch and setup the folder structure
5.      Go into the System folder.  Open HevP2ReferenceApplication.slx
6.	You can select different drive cycles in the Drive Cycle Source block.  If you have never used this block, click on Install Addtional Drive cycles and follow the prompts.  
7.   	In the Visualization subsystem, you can open the Performance and FE Scope.  Press play to watch the P2 vehicle follow the drive cycle reference.
